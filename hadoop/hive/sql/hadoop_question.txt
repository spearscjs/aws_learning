2. What is hdfs file system? Whats the benefit?
HDFS is a distributed file system on top of the hadoop architecture
HDFS function is to break the files into chunks and replicates the data
We have low cost machines to store
We have master node (NameNode) -- stores meta data, manages data nodes, 
We have data nodes which store and process data 


3. What is a block in hdfs? Default size?
A block is the smallest ammount of memory on disk that we can store. Files will be broken into this amount when stored into memory
